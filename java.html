Curso Básico de Java

¿Qué es Java?

Java es un lenguaje de programación de alto nivel que se caracteriza por ser versátil, portable y orientado a objetos. Fue desarrollado por Sun Microsystems a principios de la década de 1990 y desde entonces ha ganado popularidad debido a su capacidad para funcionar en diferentes plataformas y su amplio uso en el desarrollo de aplicaciones.

Características básicas de Java

  1. Portabilidad

    El código escrito en Java puede ejecutarse en diferentes sistemas operativos sin necesidad de reescribirlo, gracias a la máquina virtual de Java (JVM).

  2. Orientado a objetos

    Java está diseñado principalmente como un lenguaje de programación orientado a objetos, lo que significa que se enfoca en el uso de objetos y clases para organizar y estructurar el código.

  3. Seguridad

    Java incorpora diversas características de seguridad para prevenir amenazas como virus y malware. La ejecución de código no confiable se controla mediante un modelo de seguridad robusto.

  4. Multiplataforma

    La portabilidad de Java se debe en gran medida a su capacidad para ejecutarse en múltiples dispositivos y sistemas, como computadoras, dispositivos móviles, servidores, entre otros.

  5. Amplia biblioteca estándar

    Java tiene una amplia gama de bibliotecas estándar que proporcionan funcionalidades listas para usar para diversas tareas, lo que facilita el desarrollo de aplicaciones.

Clases en Java

¿Qué es una clase en Java?

Una clase en Java es como un plano o un molde que define cómo se crea un objeto. Piensa en una clase como un conjunto de instrucciones que dice cómo deben ser los objetos. Ejemplo sencillo: Si piensas en un objeto "Auto", la clase Auto definiría qué características tiene ese auto (como color, marca, modelo) y qué puede hacer (como arrancar, detenerse).

Atributos en Java

¿Qué son los atributos en Java?

Los atributos en Java son como las características o propiedades de un objeto. Son variables que representan el estado de un objeto y almacenan datos específicos sobre ese objeto. Ejemplo sencillo: En la clase Auto, los atributos podrían ser color, marca, modelo, que almacenan información como "rojo", "Toyota", "Corolla".

Métodos en Java

¿Qué son los métodos en Java?

Los métodos en Java son como las acciones que puede realizar un objeto. Son funciones definidas dentro de una clase que representan el comportamiento de los objetos. Ejemplo sencillo: En la clase Auto, los métodos podrían ser arrancar(), detenerse(), que representan acciones como "iniciar el motor" o "detenerse"

    BUCLES O CICLOS EN JAVA

    Los bucles son estructuras de control que permiten ejecutar un bloque de código repetidamente mientras se cumpla una condición específica. Los bucles son fundamentales para la programación y ayudan a automatizar tareas repetitivas. Hay tres tipos principales de bucles en Java: for, while, y do-while. Cada uno tiene sus propias características y se utiliza en situaciones particulares:

    1. Bucle for

      2. El bucle for es útil cuando se conoce el número exacto de iteraciones que se deben realizar. Se compone de tres partes:
      Inicialización: Se establece el valor inicial del contador.
      Condición: Verifica si se cumple la condición para continuar el bucle.
      Iteración: Incrementa o modifica el contador en cada iteración.

      Ejemplo:

    2. Bucle while

      3. El bucle while se utiliza cuando se desconoce el número exacto de iteraciones, pero se conoce la condición de parada al principio del bucle. Se evalúa la condición antes de cada iteración.

      Ejemplo

    3. Bucle do-while

      4. El bucle do-while es similar al while, pero garantiza que el bloque de código se ejecutará al menos una vez antes de verificar la condición de salida.

  • Encapsulamiento

    • Este principio es fundamental en la programación orientada a objetos. Consiste en ocultar los detalles de implementación de un objeto y exponer solo la interfaz necesaria para interactuar con él.

  • Abstracción

    • Se refiere a la capacidad de enfocarse en los aspectos esenciales de un objeto mientras se ocultan los detalles innecesarios.

    Ejemplo

  • Herencia

    • Es un principio que permite a una clase heredar propiedades y comportamientos de otra clase. En Java, se logra mediante la palabra clave extends. La clase que hereda se conoce como subclase o clase hija, y la clase de la que se hereda se conoce como superclase o clase padre.

  • Polimorfismo

    • Se relaciona con la capacidad de un objeto para presentar diferentes formas o comportamientos. Puede manifestarse a través de la sobrecarga y la sobrescritura de métodos.
